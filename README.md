## Notas para el GIT

1. (preparar el paquete)

```bash
git add .
```

2. Agregar la etiqueta a la caja - Mensaje del commit

```bash
git commit -m "Mensaje a especificar"
```

3. Mandar el paquete - Push pushear

```bash
git push
```

Nota: Siempre preguntarle al proyecto local el estado

```bash
git status
```
